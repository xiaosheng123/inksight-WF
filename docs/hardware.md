# InkSight 硬件指南

本文档以**当前代码仓库**为准，说明推荐硬件方案、已内置的固件构建配置、接线方式和常见硬件问题。
如果你是第一次搭建，建议优先使用 **ESP32-C3 + 4.2寸墨水屏**。
这篇适合 **准备自己动手搭硬件、选板子、对接线和供电有疑问的人**。

## 1. 推荐方案（首选）

当前推荐、也是默认固件配置的组合是：

- **主控**：ESP32-C3
- **屏幕**：4.2寸 SPI E-Paper
- **固件环境**：`epd_42_c3`

原因：

- 当前默认 `platformio` 环境就是 `epd_42_c3`
- 资料、文档、配置页预览都优先围绕 4.2寸版本展开
- 成本、体验、可复现性之间最均衡

## 2. 推荐 BOM

| 部件 | 推荐选型 | 说明 |
|------|----------|------|
| MCU | ESP32-C3 Pro mini / Super mini开发板 | 低功耗、体积小、适合首发版本 |
| Display | 4.2寸 SPI 墨水屏模块 / 裸屏（裸屏需要搭配驱板使用） | 当前默认显示规格 |
| USB | USB 数据线 | 刷机和调试必须支持数据传输 |
| 连线 | 杜邦线 / 焊线 | 原型阶段可先用杜邦线 |
| 电源 | USB 直供（开发期） | 最稳定，排障方便 |
| 锂电池（可选） | 软包 `505060-2000mAh` | 标称3.7V，需接开发板的 5V 引脚（通过板载 LDO 降压） |
| 充电（可选） | TP5000 | 默认 4.2V 充电模式即可，无需改动 |

典型 DIY BOM 仍然可以控制在 **约 220 元** 左右，但实际成本取决于屏幕采购渠道与外壳方案。

## 3. 当前代码内置的硬件构建配置

当前默认环境是 `epd_42_c3`（对应 4.2 寸黑白屏、ESP32-C3 Pro mini / Super mini 开发板），对外文档与产品流程也统一围绕 **4.2寸版本** 展开。

如果你需要查看代码中其他内置 profile，可以直接查看：

- `firmware/platformio.ini`

但如果你只是第一次搭建，建议直接使用默认的 **4.2寸方案**。

## 4. 引脚定义

当前引脚定义位于：`firmware/src/config.h`

### ESP32-C3 Profile

| 功能 | 引脚 |
|------|------|
| MOSI | `GPIO6` |
| SCK | `GPIO4` |
| CS | `GPIO7` |
| DC | `GPIO1` |
| RST | `GPIO2` |
| BUSY | `GPIO10` |
| 锂电池 ADC | `GPIO0` |
| 配置按键 | `GPIO9` |
| LED | `GPIO3` |

### ESP32-WROOM32E Profile

| 功能 | 引脚 |
|------|------|
| MOSI | `GPIO14` |
| SCK | `GPIO13` |
| CS | `GPIO15` |
| DC | `GPIO27` |
| RST | `GPIO26` |
| BUSY | `GPIO25` |
| 锂电池 ADC | `GPIO35` |
| 配置按键 | `GPIO0` |
| LED | `GPIO2` |
